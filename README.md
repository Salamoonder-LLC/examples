# Salamoonder Examples

Welcome to the Salamoonder Examples repository! This collection showcases practical examples and use cases for the Salamoonder SDK.

## About

This repository contains a curated set of examples demonstrating how to build with the Salamoonder SDKs:
- [Salamoonder Python SDK](https://github.com/Salamoonder-LLC/salamoonder-py)
- [Salamoonder JavaScript SDK](https://www.npmjs.com/package/salamoonder-js)

Whether you're just getting started or looking for inspiration on how to solve specific problems, you'll find working code samples here for both Python and JavaScript.

## Getting Started

### Python Examples

#### Prerequisites

- Python 3.7+
- The Salamoonder Python SDK

#### Installation

Clone both the Salamoonder SDK and this examples repository:

```bash
git clone https://github.com/Salamoonder-LLC/salamoonder-py.git
git clone https://github.com/Salamoonder-LLC/examples.git
```

Make sure the `salamoonder` folder from `salamoonder-py` is available locally, and you're all set to run the examples!

### JavaScript Examples

#### Prerequisites

- Node.js 12+
- npm or yarn

#### Installation

Simply install the `salamoonder-js` package:

```bash
npm i salamoonder-js
```

Or with yarn:

```bash
yarn add salamoonder-js
```

Then explore the JavaScript examples in the `javascript/` directory.

## Examples

This repository contains examples for solving various bot detection and security challenges:

- **Akamai**: Web, SBSD
- **Datadome**: Interstitial, Slider
- **Incapsula**: Reese84, UTMVC
- **Kasada**

Explore the various examples in this repository to learn how to use Salamoonder for your use cases. Each example includes clear documentation and is ready to run.
